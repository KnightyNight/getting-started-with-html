---------------------------------------------------------------------------------------------
HTML
-stands for hypertext markup language
-made of text content (what is seen), markup (what it looks like), referances to other documents (images and videos), and links to other pages
---------------------------------------------------------------------------------------------
HTML Markup
-made of opening and closing tags
-the entire thing is called an element
-Use the "b" tag to make things bold, and the "em" tag to make them italicized
---------------------------------------------------------------------------------------------
HTML Attributes
-(a href = "www.imgur.com") Javert (/a)
-"Javert" becomes a link to www.imgur.com
-(a) = the anchor
---------------------------------------------------------------------------------------------
Images
-tag is "img"
-(img src = "url" alt = "text")
-"text" is what shows up when the image doesn't load
-void tag: has no centent or ending
-image tag falls under this category
---------------------------------------------------------------------------------------------
Whitespace
-"br" tag allows for breaks in the lines, seperating them
-falls under the "void tag" category
-"p" tag makes the content into a paragraph
-"pg" tag is not a void tag
-content of the tag will render as a paragraph
---------------------------------------------------------------------------------------------
Inline/block
-"br" tag simply ends a line, so it is inline
-"p" tag makes an invisible box, and is therefore block
---------------------------------------------------------------------------------------------
Span/div
-(span) text (/span) is inline, and acts much like the "br" tag
(div) text (/div) is block, and acts a lot like the "p" tag
